Welcome to Natlink's Documentation!
=====================================

Natlink is an OpenSource extension module for the speech recognition program [Dragon](https://www.nuance.com/dragon/business-solutions/dragon-professional-individual.html).

Natlink with Python 3 and Dragon 13, 14, or 15, is now in a beta phase for Dragonfly and user defined grammars.

For Unimacro and Vocola(2) this version cannot be used yet. See the [version with Python 2.7](https://qh.antenna.nl/unimacro/installation/installationstableversionpython2.html).
